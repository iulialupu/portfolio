This is a web application where the user can register and submit a recipe. The back-end is a RESTful API server, and a database for storing the recipe and user information. The front-end part is build with React and Redux. It includes forms, with form validation for creating and editing a recipe, as well as forms for user authentication. The passwords are encrypted with Bcrypt and saved into the database. Some of the information about the users is saved in a token, using JsonWebToken, and added to the localstorage.
