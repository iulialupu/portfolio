E-commerce Clothing Store

  • React
  • Redux
  • React Router
  • Redux-persist
  • Redux-router-dom
  • Redux-thunk
  • Axios
  • CSS3
  • Json-server

The project workflow started with research and gathering information about the site structure, features as well as ideas for the visual aspect and style guides, wireframes.

The site consist of a homapage, a page showcasing all the products, with the possibility of filtering items by category, an individual product showcase page, as well as a wishlist and cart pages. Redux was used for the state management, where was stored the information about the products, fetched from the API, using Axios. Also Redux Persist helped save into the localstorage a part of the Redux store, containing information about the wishlist and cart, to make it accessible for future sessions.
